GameDev Springboard - Fall 2022 Web Design Final

Preston Corless, maintainer of GameDev Springboard.

For me, entering the Game Development world was very difficult when I was younger, as I didn't know where to start, or which path to take. I think a large part of that was simply not knowing enough about how to begin. For this reason, I began documenting the different methods to create video games for different systems. One such example was a spreadsheet containing information about which tools to use to compile or run different programming languages on different gaming systems and consoles I used growing up.

Although such information was useful, I knew this wasn't all there was, and I also knew that these means to demonstrate the information were limited. This, I set out to create a tool that improves upon this current information, makes it more accessible, and makes it easy to get started. The obvious answer was to create a web site. This has brought me to where I am today.

My research shows that nearly every publicly available modern game engine that can be reasonably found through internet searches is documented here, but note that due to the sheer number of graphics libraries and runtimes in existence, the results returned for graphics libraries are limited. There will be more additions in the future.

If you'd like to see my other projects, please check out my GitHub Pages site!
